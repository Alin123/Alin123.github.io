<1> 基本表单

code

description

  • 元素：
    1. form
      • method属性:GET,POST
      • action属性
      • enctype属性:application/x-www-form-urlencoded(默认编码格式，它不能将文件上传到服务器)、multipart/form-data(用于将文件上传到服务器)、text/plain(谨慎使用)
      • autocomplete属性
      • target属性:_blank、_parent、_self、_top、<frame>
    2. input
    3. button
      • type属性:submit(提交表单)、reset(重置表单)、button(作为一般元素使用，关联Js函数)
    4. fieldset
    5. label
  • 方法：
    1. GET(用于安全交互，同一个请求可以发起任意多次而不会产生额外作用，多用于获取只读信息)
    2. POST(则用于不安全交互，多用于会改变应用程序状态的各种操作)
